# Data_Analysis_Dashboard
## Hotel Booking Data Analysis (Excel Dashboard)

### Project Overview

This project analyzes hotel booking data using Microsoft Excel. The goal of the project is to understand booking patterns, track cancellations, and visualize important metrics through an interactive dashboard.

The dashboard helps users quickly analyze hotel booking performance using pivot tables, slicers, and charts.

### Objectives

Analyze total hotel bookings
Track booking cancellations
Understand booking trends
Create an interactive Excel dashboard for easy data visualization

### Insights
The dashboard helps identify:
Booking trends over time
Cancellation patterns
Overall hotel booking performance
